# PSA API Specifications have moved

This is no longer the upstream repository for the development the PSA API specifications.

The project is now governed by GlobalPlatform. You can find the new home for the PSA APIs at [GlobalPlatform/psa-api].

Please use that upstream repository to report [Issues][issues] and submit [Pull requests][pulls] for the PSA APIs.

[GlobalPlatform/psa-api]:  https://github.com/globalplatform/psa-api
[issues]:                  https://github.com/globalplatform/psa-api/issues
[pulls]:                   https://github.com/globalplatform/psa-api/pulls

## PSA API specifications published by Arm

This repository is a fork with the GitHub pages that hosts the [PSA API specifications] that were published by Arm until June 2026.

If you have an problem with the specification web pages, please report this by creating an [issue in this repository].

Later revisions of the specifications will be published by GlobalPlatform from a new website. Once the first specifications are published by GlobalPlatform, the pages here will be updated to direct readers to the new specifications.

[PSA API specifications]:   https://arm-software.github.io/psa-api/
[issue in this repository]: https://github.com/arm-software/psa-api/issues
